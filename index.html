Flatpak Automatic

Welcome to the official repository for Flatpak Automatic. This project provides a secure, configurable, and systemd-native automation wrapper for Flatpak updates with Snapper snapshot integration.

📦 Installation (RPM-based)

Run the following commands to add the repository and install the package on Fedora or compatible RPM-based systems:

sudo tee /etc/yum.repos.d/flatpak-automatic.repo <<'EOF' [flatpak-automatic] name=Flatpak Automatic - Stable baseurl=https://fedoraBee.github.io/flatpak-automatic/rpms/latest/stable/ enabled=1 gpgcheck=1 gpgkey=https://fedoraBee.github.io/flatpak-automatic/gpt.key [flatpak-automatic-testing] name=Flatpak Automatic - Testing baseurl=https://fedoraBee.github.io/flatpak-automatic/rpms/latest/testing/ enabled=0 gpgcheck=1 gpgkey=https://fedoraBee.github.io/flatpak-automatic/gpt.key EOF sudo dnf makecache sudo dnf install flatpak-automatic

📦 Installation (Debian-based)

Run the following commands to add the repository and install the package on Ubuntu, Debian, or compatible systems:

curl -fsSL https://fedoraBee.github.io/flatpak-automatic/gpt.key | sudo gpg --dearmor -o /usr/share/keyrings/flatpak-automatic-archive-keyring.gpg echo "deb [signed-by=/usr/share/keyrings/flatpak-automatic-archive-keyring.gpg] https://fedoraBee.github.io/flatpak-automatic/debs/latest/stable /" | sudo tee /etc/apt/sources.list.d/flatpak-automatic.list sudo apt-get update sudo apt-get install flatpak-automatic
